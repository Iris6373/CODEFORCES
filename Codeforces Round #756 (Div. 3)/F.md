## My solution :  (Accepted)

     Since we need to find the longest sequence, we can two-pointers to solve this problem.
     First, we check upto which point, the balance >= 0.
     If < 0, we change x and y if j-i > ans.
     
     Then, we increment i and check if the current balance + v[j] >= 0.
     If, so we break from current loop.
     Else, we continue unitl j < n.

              cin >> n >> s;
              vt v(n);

              i = 0, j = 0, k = 0;
              while (j < n) {
                  while (j < n && s+v[j] >= 0) s += v[j], j++;
                  if (ans < j-i) {
                      ans = max(j-i,ans);
                      x = i;
                      y = j;
                  }
                  // cout << i << " " << j << "\n";
                  while (i <= j && j < n) {
                      s -= v[i], i++;
                      if (s+v[j] >= 0) break;
                  }
